# 2️⃣ 🌍 Cómo obtienen tu IP

Laboratorio educativo en entorno controlado: se usa mi propia IP, correctamente ocultada en las capturas.  
Objetivo: entender cómo se filtra una IP al visitar ciertos enlaces.
